# Kivy Face Recognition App

This is a Python-based Face Recognition mobile application built using **Kivy** and **Buildozer**.

## Features
- Face detection and recognition
- Android APK build using Buildozer
- Python + Kivy framework

## Tech Stack
- Python
- Kivy
- OpenCV
- Buildozer
- Android SDK/NDK

## How to Build APK
```bash
buildozer android debug
